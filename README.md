# directory index を表示するだけのやつ

PHP -SがDirectory indexしてくれなくて不便なので

![ss](ss.png)

## usage

``` 
$ cd /tmp/
$ git clone git@github.com:uzulla/php_directory_index.git
$ cd /path/to/as_you_like
$ php -S 0.0.0.0:8080 /tmp/php_directory_/index.php
```
